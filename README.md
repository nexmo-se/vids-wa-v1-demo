# vids-wa-v1-demo

```js
// clone repo

// use a local tunnel app E.g. Ngrok
ngrok http 5000

mv env-example .env
// populate environment variables
// copy private.key to root of repository

npm install
npm install nodemon concurrently --save-dev

// to run both backend and frontend concurrently:
npm run dev
```
