# vids-wa-v1-demo

```js
mv env-example .env

// copy private.key to backend

cd fronend
npm install

cd backend
npm install express cookie-parser morgan dotenv jsonwebtoken js-base64 axios request
npm install nodemon concurrently --save-dev

// to run both backend and frontend concurrently:
cd backend
npm run dev
```
